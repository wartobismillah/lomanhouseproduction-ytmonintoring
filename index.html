Setup awal — cukup sekali

1
Buka Google Sheets baru → klik menu Extensions → Apps Script → hapus kode yang ada → paste kode berikut → klik Save (💾):
2
Klik Deploy → New deployment → klik ikon ⚙️ → pilih Web app → isi description bebas → Execute as: Me → Who has access: Anyone → klik Deploy → copy URL yang muncul
3
Paste URL di bawah + buat password tim → klik tombol Simpan. File index.html akan ter-download otomatis — file inilah yang di-deploy ke GitHub Pages, bukan file yang sedang Anda buka ini.
ℹ️ Alur singkat: Isi form ini → Download index.html → Upload ke GitHub Pages → Buka URL GitHub Pages → Selesai, bisa dipakai tim.
